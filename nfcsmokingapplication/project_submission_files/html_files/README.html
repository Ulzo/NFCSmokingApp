The Motivation & Goal of This Project

The motivation of this project was provided by John O'Keefe, a Data Manager working with UIC's Institute for Health Research & Policy on a study of the “Social and Emotional Contexts of Adolescent Smoking Patterns”.

For several years, Mr. O'Keefe's team has been lending Palm Pilots to adolescent subjects. These Palm Pilots contain a survey that randomly appears throughout the day; subjects answer questions about their recent smoking behavior, other behaviors they engage in while smoking, and other details about the context of their smoking behaviors. Researchers load the survey onto the Palm Pilots, and subjects return the Palm Pilots to the researchers after several days or weeks of carrying them around. The survey results are stored in files on the Palm Pilots, which the researchers then download so they can analyze the results. When the researchers want to change the survey, they must collect the Palm Pilots from subjects and load a new survey file onto the devices.

Mr. O'Keefe's idea is that, with the explosion in popularity of smartphones with continual internet connectivity, our team could try to create a proof-of-concept survey for Android phones. Our goal was to create a survey application for Android phones that reproduces the main features of the Palm Pilot version of the survey.

Instead of random survey prompts throughout the day, we thought that we could use the Near Field Communication (NFC) capabilities of newer Android phones to trigger a survey prompt soon after the user finishes smoking. The subject taps their phone against a nearby NFC tag stored on their lighter, cigarette pack, or other portable smoking tool, and then the survey prompt appears on their phone so that the user can provide details about their smoking experience. Multiple NFC tags may be activated so that the user can easily trigger the survey application under various circumstances and locations.

Folders and Files in the Zip File

The following folders and their contents are in the zip file:

  • “html_files”: This folder contains the required project hmtl files – README.html, Design.html, and INSTRUCTIONS.html. This folder also contains various files, such as pictures and screenshots, that are or were used in the creation of the html files.

  • “NFCSmokingApplicationFinal”: This folder is an Eclipse Java folder (and can be run as a project using the Android emulator within Eclipse, as is explained in INSTRUCTIONS.html). It contains the code files for the project.

    • “John_survey.xml”: The XML file of the sample survey questions given to the team by John O'Keefe.
